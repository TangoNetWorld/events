# TangoNet World

**TangoNet World** is an interactive globe that displays tango events from around the world on a 3D Earth.

Explore festivals, marathons, encuentros, workshops, and other tango-related events by navigating the globe and discovering what's happening in different countries and cities.

## Purpose

TangoNet World was created to help tango dancers, teachers, organizers, and enthusiasts discover international tango events more easily and visually.

## Data

Event information is collected from publicly available sources and is displayed for informational purposes.

## Related Project

TangoNet World is part of the [TangoNet](https://apps.apple.com/tr/app/tangonet/id6743403721?l=tr) ecosystem, a platform designed to help the tango community find, share, and promote tango events worldwide.

## Disclaimer

While efforts are made to keep event information accurate and up to date, event details may change. Please verify information with the official event organizers before making travel or registration plans.
